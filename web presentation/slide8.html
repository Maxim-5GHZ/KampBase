4. Стек технологий (Взрослый подход)

  • Backend: Java + Spring Boot. (Монолит, готовый к распилу).
  • Runtime: Axiom JDK — выбрали отечественный стандарт, чтобы показать готовность к импортозамещению и финтех-нагрузкам (привет, Т-Банк!).
  • Frontend: Next.js (React + TypeScript). Адаптирован под TMA.
  • SDK: Собственная TS-библиотека для общения фронта с бэком.
  • AI & Docs Generator: Python-микросервис + LLM + Headless LibreOffice для идеальной генерации docx в PDF без поехавшей верстки.
