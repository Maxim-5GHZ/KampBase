7.Путь Студента (CJM)

Из одинчества в дружное комьюнити .

🚪 1. Онбординг

Действие пользователя Студент авторизуется на платформе, выбирает профильную "Стихию" (например, Java) и видит свой пустой Дашборд.
Точка контакта (UI) React SPA: Login Page & Dashboard
Эмоции / Мысли 🤩 Вау, выглядит как IDE, а не сайт универа!
Магия под капотом (Система) Java Spring: Генерация JWT, создание роли ROLE_STUDENT в MariaDB.

📚 2. Поиск знаний (Wiki)

Действие пользователя Ищет "Как настроить Spring Security". Читает топовую статью, написанную старшекурсником. Жмет "Поставить Звезду" (лайк).
Точка контакта (UI) Wiki Search & Markdown Reader
Эмоции / Мысли 💡 Наконец-то понятный гайд, а не методичка 2004 года.
Магия под капотом (Система) Axiom JDK API: Умный поиск с ORDER BY stars DESC. Начисление репутации автору статьи (Транзакция БД).

⚔️ 3. Вызов (Квест-Борд)

Действие пользователя Берет практическую задачу "Написать REST контроллер". Пишет код, делает Pull Request на GitHub и прикрепляет ссылку в систему на проверку.
Точка контакта (UI) Task Board -> Submit Form
Эмоции / Мысли 😬 Надеюсь, ментор не сильно заругает мой код.
Магия под капотом (Система) POST /submissions: Создание записи в БД со статусом PENDING. Уведомление менторам.

🧙‍♂️ 4. Ревью Ментора (Core Feature)

Действие Ментора Ментор открывает код, проверяет. В панели двигает ползунки: Дать +30 XP за навык "Java" и +50 Звезд Репутации. Оставляет коммент и жмет "Одобрить".
Точка контакта (UI) Mentor Review Dashboard (Sliders)
Эмоции Студента (Уведомление) 🥳 Ура! Level Up! Плюс фидбек от реального сеньора.
Магия под капотом (Система) ACID транзакция: UPDATE user_skills, UPDATE users_balance. Статус APPROVED.

🏆 5. Триумф (HR & Награды)

Действие пользователя Студент видит себя в ТОП-5 Лидерборда по Java. Получает инвайт на стажировку от HR Т-Банка. На заработанные звезды покупает мерч (худи) в магазине вуза.
Точка контакта (UI) Public Leaderboard & Rewards Shop
Эмоции / Мысли 😎 Я готов к реальной работе. Мой профиль говорит сам за себя.
Магия под капотом (Система) Сложный SQL-запрос для агрегации и ранжирования подтвержденных скиллов. B2B вид для HR.
