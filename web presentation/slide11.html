11. План разработки MVP (30 часов)

План разработки на 30 часов. Разделение архитектуры: Backend, Frontend и Infra.

Фаза 1: Архитектура и База

Часы 0 - 5
🐧 SysAdmin (Денис)
  • Аренда и базовая подготовка VPS (Linux).
  • Установка Docker Engine и Docker Compose.
  • Написание `docker-compose.yml` для БД (MariaDB).
  • Infra: Linux Server, Docker, MariaDB
⚙️ Backend (Максим)
  • Развертывание Java Spring Boot на Axiom JDK.
  • Проектирование схемы БД (Users, Skills).
  • Настройка Spring Security и JWT авторизации.
  • API: POST /login, POST /register
🎨 Frontend (Костя)
  • Инициализация Next.js проекта.
  • Настройка Tailwind CSS и UI-библиотеки (Shadcn).
  • Верстка Layout, навигации и страницы логина.
  • UI: Dark Mode Theme, Auth Forms

Фаза 2: База Знаний (Wiki) и Сеть

Часы 5 - 15
🐧 SysAdmin (Денис)
  • Создание единой сети Docker (`studbaza-net`).
  • Настройка Nginx как Reverse Proxy.
  • Проброс маршрутов: `/` на Next.js, `/api` на Spring Boot.
  • Infra: Nginx.conf, Docker Network, CORS Config
⚙️ Backend (Максим)
  • CRUD для статей Wiki.
  • Умный поиск статей (ORDER BY stars DESC).
  • Логика начисления рейтинга авторам.
  • API: GET /articles, POST /articles/{id}/star
🎨 Frontend (Костя)
  • Страница поиска статей и выдачи результатов.
  • Интеграция Markdown-ридера для чтения.
  • Микроинтеракции: анимация кнопки "Поставить звезду".
  • UI: Wiki Board, Article View

Фаза 3: Квесты, Ревью и Релиз в Web

Часы 15 - 25
🐧 SysAdmin (Денис)
  • Привязка доменного имени (настройка DNS A-записей).
  • Получение SSL-сертификатов через Certbot.
  • Перевод Nginx на порт 443 (HTTPS), редирект с 80-го.
  • Infra: Domain, Let's Encrypt SSL, HTTPS
⚙️ Backend (Максим)
  • Логика сдачи тасок (Submissions).
  • ACID-Транзакция: обновление XP по навыкам и начисление Звезд.
  • Генерация HR-Лидерборда.
  • API: POST /tasks/{id}/review, GET /leaderboard
🎨 Frontend (Костя)
  • Дашборд Студента (Аватар, Шкалы скиллов).
  • Кабинет Ментора: формы и ползунки (Sliders) выдачи опыта.
  • Публичная таблица рейтинга для HR.
  • UI: Mentor Review Form, Radar Charts

Фаза 4: Интеграция и Питч

Часы 25 - 30
🚀 Командная работа
  • Сборка всех контейнеров в едином docker-compose up -d --build (Денис).
  • Связка Frontend и Backend по реальному HTTPS домену (Костя + Максим).
  • Сквозное тестирование: Студент -> Сдача -> Одобрение Ментором -> Рост скилла в Лидерборде.
  • Подготовка демо-сценария и питч-презентации.
