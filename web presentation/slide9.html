Архитектурная схема и MVP на 30 часов

Next.js Front -> TS SDK -> Java Spring Backend -> DB (PostgreSQL) / GitHub API / AI Microservice.

Что мы реально закодим за хакатон (MVP):

  1. Микросервис авторизации (JWT сессии).
  2. API для Wiki (CRUD: поиск, добавление, удаление).
  3. Логику "Check Access" (валидация баланса звезд для доступа).
  4. Webhook listener для GitHub (прием пушей и начисление опыта).
  5. Интерфейс Next.js App.
